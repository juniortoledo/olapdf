# PHP - Kit
## _Minimum development kit_


## Features

- MVC
- Routes
- ORM
- Image uploading
- Api

## Installation

```sh
git clone https://github.com/juniortoledo/php-kit/
```
```sh
cd php-kit
```
```sh
composer update
```
